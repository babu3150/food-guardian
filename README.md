# food-guardian（食材の番人）
https://food-guardian.net

## これは何？

### 食材の賞味（消費）期限を管理するアプリ
* 食材名と賞味（消費）期限を登録することで、冷蔵庫にある食材を可視化するWebアプリケーション
* お試し用ユーザーあり
  * ユーザー名が「testuser」、パスワードが「p4wk」

## 動作環境
* 最新バージョンのWebブラウザーであれば問題なく動作するはず
  * Google Chrome、Microsoft Edge、Firefoxで動作確認済み
* スマホ、タブレット、パソコンで使用可能

## 各画面の説明

### ログイン画面
* ユーザー名、パスワードが必須入力
* ユーザー登録が済んでいない場合は「冷蔵庫の主でない者はこちら」からユーザー登録をおこなう
* ログイン後は食材リスト画面に遷移する

### ユーザー登録画面
* ユーザー名、パスワードを入力し、登録をおこなう
* すでに登録されているユーザー名は登録不可、パスワードはなんでもよい
  * ユーザー名、パスワードともに文字数、文字の種類等の制限なし
* ログイン画面に戻りたい場合は「冷蔵庫の扉を開く者はこちら」をクリック
* ユーザー登録後は自動的にログインがおこなわれ、食材リスト画面に遷移する

### 食材リスト画面
* メインページ
* 食材登録フォーム、登録済みリスト、メモエリアの3つで構成
* 食材登録フォームでは食材名、期限を入力し、登録をおこなう
* 登録済みリストには登録した食材とその期限が一覧表示される
  * 期限が近い（古い）ものから順に表示し、期限切れの食材は期限を赤色、期限当日の食材は期限を黄色で表示
* 食材を冷凍した場合は冷凍ボタン、消費した場合は削除ボタンをクリック
  * 冷凍した食材はリストの最後に表示される
* メモには自由に文字を入力可能、保存ボタンを押して保存可能
* ヘッダーにはAI問い合わせ画面へのリンクボタンとログアウトボタンがある

### AI問い合わせ画面
* カテゴリーをタグボタンの中から１つ選択し、食材名を入力して質問することでAIが回答してくれる
* 食材名以外を入力してもなにかしら回答してくれる
* ヘッダーには食材リスト画面へのリンクボタンとログアウトボタンがある
